## 📌 Overview

This project predicts whether a customer is likely to churn using an Artificial Neural Network (ANN). It includes a machine learning pipeline and an interactive web interface built with Streamlit.

---

## 🚀 Features

- Predict customer churn in real-time
- Interactive user interface
- ANN-based model (MLP Classifier)
- Data preprocessing and feature engineering
- Clean and simple deployment using Streamlit Cloud

---

## 🧠 Model Details

- Model Type: Artificial Neural Network (ANN)
- Implementation: Scikit-learn (MLPClassifier)
- Architecture:
  - Hidden Layer 1: 64 neurons (ReLU)
  - Hidden Layer 2: 32 neurons (ReLU)
- Output: Binary classification (Churn / Not Churn)

---

## 📊 Key Insights

The most influential factors affecting churn include:

- Number of products
- Account balance
- Customer activity status
- Age
- Geography

Behavioral features were found to be more impactful than financial attributes.
